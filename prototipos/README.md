# Protótipos

Esta pasta reúne experimentos que não fazem parte do fluxo principal do site.

- `jogo-basico.html`: primeiro teste de interação narrativa e rolagem de D20;
- `d20-3d/`: experimento visual com Three.js.

Manter os protótipos separados evita que arquivos exploratórios sejam confundidos com módulos ativos do RPG Solo. Antes de integrar qualquer experimento ao produto, revise sua estrutura, acessibilidade e dependências.
